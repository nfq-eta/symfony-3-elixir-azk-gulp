Collected buttons

This button is "collected" from another module (random number).

Module files will load only when you click on it. Button functionality uses service from that module.

Some things can still be loaded before other files - for example, button text is translated using Angular service.

Translations

Translations are dumped using BazingaJsTranslationBundle (on composer install).

They can be loaded before loading some module.

For example, button text is translated. To see it in Lithuanian, switch language.
